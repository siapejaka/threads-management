Integrasi n8n (Fast Scheduling)

Untuk memicu pengecekan jadwal postingan setiap 1 menit via n8n:

  1. Buat node Schedule Trigger di n8n (Atur interval ke 1 menit).
  2. Sambungkan ke node HTTP Request.
  3. Atur Method ke GET.
  4. Gunakan URL Web App GAS Anda dengan format parameter berikut:
URL_WEB_APP_ANDA/exec?action=run_cron&token=threads_cron_secret_123
